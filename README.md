## Test Automation Project

Проект автоматизации тестирования для интернет-магазина [ФК "Краснодар"](https://shop.fckrasnodar.ru/)

---

### Стек технологий

- ***Java***
- ***Maven***
- ***JUnit 5***
- ***Selenium Webdriver***
- ***Rest Assured***
- ***AssertJ***
- ***Gson***

---

### Тестируемый функционал
#### UI

- Главная страница
- Форма логина
- Форма регистрации
- Восстановление пароля
- Поиск
- Корзина

#### API

- Форма логина
- Поиск
